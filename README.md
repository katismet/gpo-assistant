# ГПО-Помощник
AI-система управления строительными сменами  
Telegram-бот + LLM-анализ + интеграция с Bitrix24

## Описание проекта
ГПО-Помощник — это AI-ассистент для автоматизации управленческого учёта в строительстве.
Система предназначена для сбора плановых и фактических данных по сменам, анализа эффективности,
формирования отчётов и получения управленческих инсайтов в режиме реального времени.

## Бизнес-задача
- разрозненный ручной учёт данных по сменам;
- отсутствие оперативной управленческой картины;
- потери времени и денег из-за человеческого фактора;
- сложность формирования отчётности.

## Решение
Telegram-бот, интегрированный с Bitrix24 и AI-модулем, который:
- собирает данные по сменам;
- анализирует план/факт показатели;
- формирует отчёты и управленческие выводы;
- выявляет отклонения и риски.

## Основной функционал
- Ввод плановых и фактических данных по смене
- Учёт ресурсов и табеля
- AI-анализ эффективности
- Генерация текстовых отчётов и инсайтов для руководства

## Используемые технологии
- Python
- Telegram Bot API
- OpenAI API (LLM)
- Bitrix24 REST API
- Backend-разработка в Cursor

## Архитектура
Telegram → Backend-сервис → Bitrix24  
                        ↘ AI-модуль (LLM)
Схема архитектуры: `docs/architecture.png`

## Статус проекта
Проект реализован в виде MVP и готов к практическому использованию.
